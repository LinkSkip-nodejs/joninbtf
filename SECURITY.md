Security Policy
Supported Versions

This section lists the versions of the project that currently receive security updates.

Version	Supported
5.1.x	✅ Yes
5.0.x	❌ No
4.0.x	❌ No
< 4.0	❌ No

Only supported versions will receive timely security patches. Versions marked as unsupported may contain vulnerabilities and should be upgraded.

Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it responsibly.
Where to report: Send a detailed description to my Discord: linky_dabest.

Response time: We aim to acknowledge all reports within 48 hours.

Resolution updates: You will be kept informed on the progress until the issue is fixed or deemed not applicable.

Responsible disclosure: Please do not publicly disclose the vulnerability until a fix is released.

We appreciate responsible reporting and will credit contributors for any verified security issues that are resolved.
